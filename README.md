# SmartCart Clustering System (Unsupervised Machine Learning)

## Overview

The SmartCart Clustering System is an Unsupervised Machine Learning project that analyzes customer data and groups customers into different segments based on their purchasing behavior and demographic characteristics.

Customer segmentation helps businesses understand different types of customers and improve marketing strategies.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook
* VS Code

## Project Workflow

Customer Dataset
↓
Data Cleaning
↓
Feature Engineering
↓
Feature Scaling
↓
Clustering (K-Means Algorithm)
↓
Customer Segmentation
↓
Cluster Visualization

## Clustering Algorithm

The project uses **K-Means Clustering**, an unsupervised learning algorithm used to group similar customers into clusters.

## Features Used

* Age
* Income
* Education
* Marital Status
* Total Spending
* Customer Tenure
* Number of Children
